Problem 1: Standard Linear Program (Maximization)

Given
c = [2, 3]
A = [[1, 2],
     [3, 1]]
b = [8, 9]

Solve for x ∈ R^2:

maximize    c^T x

subject to  A x <= b
            x >= 0


Problem 2: Standard Linear Program (Minimization)

Given
c = [4, 1, 3]
A = [[1, 1, 0],
     [2, 0, 1],
     [0, 1, 2]]
b = [5, 6, 8]

Solve for x ∈ R^3:

minimize    c^T x

subject to  A x >= b
            x >= 0


Problem 3: Linear Program with Equality Constraints

Given
c = [1, 5, 2]
Aeq = [[1, 1, 1],
       [2, -1, 1]]
beq = [7, 4]

Solve for x ∈ R^3:

minimize    c^T x

subject to  Aeq x = beq
            x >= 0


Problem 4: Linear Program with Lower and Upper Bounds

Given
c = [3, 1, 4]
A = [[1, 1, 1],
     [2, 1, 0]]
b = [10, 12]
l = [1, 0, 2]
u = [5, 4, 6]

Solve for x ∈ R^3:

maximize    c^T x

subject to  A x <= b
            l <= x <= u


Problem 5: Mixed Equality and Inequality Constraints

Given
c = [2, 1, 3, 2]
A = [[1, 2, 0, 1],
     [0, 1, 1, 2]]
b = [10, 8]
Aeq = [[1, 1, 1, 1]]
beq = [7]

Solve for x ∈ R^4:

minimize    c^T x

subject to  A x <= b
            Aeq x = beq
            x >= 0


Problem 6: Quadratic Program (Convex)

Given
Q = [[4, 1],
     [1, 2]]
c = [-8, -6]
A = [[1, 1],
     [1, 0],
     [0, 1]]
b = [5, 4, 4]

Solve for x ∈ R^2:

minimize    (1/2) x^T Q x + c^T x

subject to  A x <= b
            x >= 0


Problem 7: Constrained Least Squares

Given
A = [[1, 2],
     [2, 1],
     [1, 1]]
b = [4, 3, 2]

Solve for x ∈ R^2:

minimize    ||A x - b||_2^2

subject to  x1 + x2 <= 3
            x1 >= 0
            x2 >= 0


Problem 8: L1 Optimization

Given
A = [[1, 1],
     [1, -1],
     [2, 1]]
b = [3, 1, 4]

Solve for x ∈ R^2:

minimize    |x1| + |x2|

subject to  A x = b


Problem 9: Portfolio-Style Quadratic Program

Given
Sigma = [[0.10, 0.02, 0.04],
         [0.02, 0.08, 0.01],
         [0.04, 0.01, 0.12]]
r = [0.12, 0.10, 0.15]

Solve for x ∈ R^3:

minimize    x^T Sigma x

subject to  r^T x >= 0.12
            1^T x = 1
            x >= 0


Problem 10: Chebyshev-Type Minimax Optimization

Given
a1 = [1, 2],   b1 = 5
a2 = [2, -1],  b2 = 1
a3 = [1, 1],   b3 = 4

Solve for x ∈ R^2 and t ∈ R:

minimize    t

subject to  |a1^T x - b1| <= t
            |a2^T x - b2| <= t
            |a3^T x - b3| <= t
